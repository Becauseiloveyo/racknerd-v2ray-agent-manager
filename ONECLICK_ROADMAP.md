# 小白一键安装脚本设计路线

这个文件用来记录后续脚本升级方向。

目标不是把所有开源脚本混装到一台 VPS，而是吸收它们的好设计：安装前检测、自动备份、端口选择、失败提示、服务管理、平台检测、客户端提示。

## 基本原则

1. 少协议，少变量。
2. 优先稳定和可排查。
3. 不承诺 100% 解锁 AI / 影视。
4. 不把机房 VPS IP 伪装成住宅 IP。
5. 不直接复制其他项目代码，参考思路时保留来源和许可证。

## 推荐主方案

```text
Xray-core
VLESS
Reality
Vision
TCP
443 优先
IPv4 优先
Mux 关闭
fingerprint: chrome
```

原因：

- 协议少，方便排查。
- 443 对手机热点、宽带、公司网、校园网兼容性通常更好。
- Reality 不依赖域名证书，适合没有域名或不想维护证书的小 VPS。
- Vision 是当前常见的 VLESS Reality 搭配方式。

## 参考开源项目时吸收的点

### Xray-core

吸收点：

- 使用官方 release 获取最新核心。
- 使用 Xray 自带命令生成 UUID 和 Reality x25519 密钥。
- 安装前做 config test。
- systemd 管理服务。

### v2ray-agent

吸收点：

- 菜单式交互。
- 多项检测和修复。
- 端口管理。
- 备份配置。
- 小白提示。

不吸收点：

- 一次安装太多协议。
- 过多复杂分流和回落逻辑。

### sing-box

吸收点：

- 统一配置思路。
- DNS 和路由策略清晰化。
- IPv4 / IPv6 策略可控。

暂不默认安装：

- 避免 Xray 和 sing-box 双核心并存造成排查困难。

### Hysteria2 / TUIC

吸收点：

- 作为备用线路方案。
- UDP/QUIC 网络好的时候速度好。

暂不默认安装：

- 部分运营商或热点对 UDP 不稳定。
- 对小白来说和 Reality 同时安装会增加排查成本。

### 3x-ui / Hiddify / Marzban 类面板

吸收点：

- 页面化管理。
- 用户和节点展示清楚。
- 状态和日志入口明显。

暂不默认安装：

- 面板会增加暴露面和维护成本。
- 1GB 内存小 VPS 不适合堆太多常驻组件。

## 一键脚本应具备的功能

### 安装前

- 检查系统版本。
- 检查 root 权限。
- 检查内存、磁盘、架构。
- 检查端口占用。
- 检查 GitHub 可访问性。
- 检查系统时间。
- 检查 DNS。
- 自动备份旧配置。

### 安装中

- 安装基础工具。
- 修复 DNS。
- 开启 NTP。
- 开启 BBR。
- 自动选择端口，优先 443。
- 安装最新 Xray-core。
- 生成 Reality 参数。
- 写入最小可用配置。
- 配置 systemd。
- 启动前做配置测试。

### 安装后

- 输出节点参数。
- 输出 VLESS 导入链接。
- 输出二维码。
- 安装管理命令，例如 `rnai`。
- 自动检测 AI / 影视平台连通性。
- 给出客户端设置建议。

## 管理菜单设计

```text
╔════════════════════════════════════╗
║      RackNerd AI Reality 管理      ║
╚════════════════════════════════════╝

1. 服务状态        看 Xray 是否运行
2. 显示节点        链接、二维码、参数

4. 查看日志        失败时排查

6. 一键修复        DNS/时间/防火墙/服务

8. 备份配置        备份当前节点
9. 更新核心        更新 Xray-core
0. 退出
```

## AI / 影视检测说明

检测对象：

```text
ChatGPT
OpenAI
OpenAI API
OpenAI 静态资源
Grok
xAI
X
YouTube
Netflix
Disney+
Prime Video
TikTok

```

检测结果只表示 VPS 出口能否连接这些站点，不等于账号、地区、片库或功能一定解锁。

## 对“全部解锁”的处理方式

脚本可以做：

```text
优化协议
优先 443
IPv4 优先
修复 DNS
修复系统时间
减少 DNS 泄漏
检测平台响应
给出失败原因
提示换 IP / 换机房 / 换住宅出口
```

脚本不能做：

```text
保证任何平台都放行
保证任何运营商都稳定
把 RackNerd 机房 IP 变成住宅 IP
绕过账号地区、支付地区、设备环境风控
```

## 推荐实施顺序

### 第一步

保留现有 `fresh_setup.sh`，作为稳定初始化入口。

### 第二步

新增 `ai_reality_oneclick.sh`：

```text
自动安装 Xray Reality Vision 443
自动生成节点链接和二维码
自动安装 rnai 管理菜单
```

### 第三步

把主菜单接入：

```text
31. 小白一键安装 AI/影视优化 Reality 节点
```

### 第四步

增加回滚和卸载：

```text
备份旧配置
恢复旧配置
卸载新方案
```

### 第五步

增加检测报告：

```text
系统信息
IP ASN
DNS
时间
端口
Xray 状态
AI/影视 HTTP 状态
客户端建议
```
